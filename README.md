# Customer Sales Analysis

This project analyzes customer purchasing behavior using Python and Power BI.

Tools Used:
- Python
- Pandas
- Matplotlib
- Power BI

Key Insights:
- Kavya Khan is the top purchasing customer
- Electronics generated the highest revenue
- Kolkata generated the highest city sales
- Card payment method is mostly used by customers

Dashboard includes:
- Sales by Category
- Sales by City
- Payment Method Distribution
- Top Customers
